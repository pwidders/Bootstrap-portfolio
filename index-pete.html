About

Hello! Throughout my life, I've been a student, a military soldier and a salesman with a family background that instilled the virtues of a hard work-ethic, kindness, honor and respect for people of all backgrounds. Most of my professional career was based in sales and customer-service, however I've recently found a love for web development and I'm working diligently to complete a full-stack coding bootcamp at the UC Berkeley Extension and pursue a career in this field! I love building things and problem-solving. I also hold a bachelor's degree in communications from California State University East Bay. Aside from my professional activities, I'm proud to say I'm recently engaged to my beautiful fiance Hannah and we have a Boston Terrier/French Bulldog mix dog named Rico. They are both awesome and occupy what little free time I have these days.

In the Army, I was a 13M field-artilleryman. I had a chance to see more of the world making stops in Ft. Sill, OK, South Korea and Iraq. Some of my favorite places include Hawaii (Maui), Lake Tahoe, Lake Shasta and tailgating at the Oakland Coliseum. One can still hope!

Code Projects

Travelbook

An app for travelers to document their interests and recommendations. This was my first experience working with a group tasked to 'build something awesome.' We designed the app to be a social media type personal profile app that allowed users to network with friends and share their travels. Technologies used were HTML, CSS, Javascript, Firebase, Yelp API and Google developer. The app is continually updating.

Giftoonopia

A fun cartoon-themed application using Giphy API. This was an assignment to get practice using the ajax method to query API's. The interface allows users to click any of the prearranged cartoons, or enter new cartoons of their choosing to pull up a slate of randomly generated gifs that match that theme. Technologies used were HTML, CSS and Javascript with Giphy API.


Train time!

This was an exercise utilizing Firebase to log train times and give realtime departure times. The interface allows users/admins to log train data that gets updated in real time to the page. Technologies used were HTML, CSS, Javascript, Momento.js, Firebase.

Western word scramble

This was my first Javascript assignment, which was to build a hangman-style game. I chose a Western-style theme. It was built using HTML, CSS and vanilla Javascript. The interface allows the user to click a start game button, which then generates underscores that represent the hidden word to guess.


Crystals Collector

This was a game with an object of clicking on one of four buttons that would return an assigned hidden value, which then got added to your total score. The point of the game is to try and match your score with the random number chosen at the start. If you go over, you lose. If you hit it, you win! This app was built with HTML, CSS and Javascript with a focus on Jquery.
