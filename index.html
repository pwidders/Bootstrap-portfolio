Giftoonopia

A fun cartoon-themed application using Giphy API. This was an assignment to get practice using the ajax method to query API's. The interface allows users to click any of the prearranged cartoons, or enter new cartoons of their choosing to pull up a slate of randomly generated gifs that match that theme. Technologies used were HTML, CSS and Javascript/jQuery with Giphy API.

Github Deployed

Train time!

This was an exercise utilizing Firebase to log train times and give realtime departure times. The interface allows users/admins to log train data that gets updated in real time to the page. Technologies used were HTML, CSS, Javascript, Momento.js, Firebase.

Github Deployed

Crystals Collector

This was a game with an object of clicking on one of four buttons that would return an assigned hidden value, which then got added to your total score. The point of the game is to try and match your score with the random number chosen at the start. If you go over, you lose. If you hit it, you win! This app was built with HTML, CSS and Javascript with a focus on Jquery.

Github Deployed

Liri Node App

An app that utilizes Spotify, OMDB and Bands in town to perform search queries. Functions include 'Spotify It' - search Spotify's API to find information by song, 'Movie It' - search OMDB database by movie title and 'Concert It' - search Bands in Town database to see if an artist is on tour. Technologies include Node.js with Inquirer, Moment and request packages, as well as Javascript.

Github
